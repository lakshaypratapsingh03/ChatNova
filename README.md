# 📚 ChatNova – AI Study Assistant

## 🚀 Project Overview

ChatNova – AI Study Assistant is a full-stack AI-powered web application designed to help students with their academic tasks through intelligent text-based interaction. The system provides real-time answers, explanations, and coding assistance using advanced Natural Language Processing (NLP).

The platform simplifies learning by acting as a smart assistant that can answer queries, generate explanations, and support self-learning in an interactive way.

---

## 🎯 Objectives

* Provide instant and accurate answers to user queries
* Assist students in coding and problem-solving
* Generate notes, summaries, and explanations
* Ensure secure authentication and data management
* Develop a scalable and user-friendly application

## 🏗️ System Architecture

The system follows a layered architecture:

1. **User Interface Layer** – Chat interface for user interaction
2. **Application Layer** – Backend server handling logic
3. **AI Processing Layer** – Processes queries using NLP
4. **Database Layer** – Stores user data and chat history
5. **External Services** – AI APIs and cloud services

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/chatnova.git
cd chatnova
```

### 2. Install dependencies

```
npm install
cd client
npm install
```

### 3. Setup environment variables

Create a `.env` file in server folder:

```
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
GEMINI_API_KEY=your_api_key
```

### 4. Run the project

```
npm run dev
```
