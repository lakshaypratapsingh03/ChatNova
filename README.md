# 🤖 ChatNova AI

ChatNova AI is a full-stack AI powered chat application that allows users to interact with artificial intelligence for **text generation and image generation**. The platform is built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)** and integrates external AI services like **Google Gemini API** and **ImageKit API**.

The application provides a modern UI with features like **dark mode, credit-based usage system, AI chat, AI image generation, and community images gallery**.

---

# 🚀 Features

### 💬 AI Text Generation
Users can ask questions or prompts and receive intelligent responses generated by AI using **Gemini API**.


### 💳 Credit System
Users receive credits to generate text and images.

### 👤 Authentication
- User Signup / Login
- Google Authentication support

### 🌙 Dark Mode
The application supports **light and dark theme** for better user experience.



### 📊 Credit Plans
Users can purchase additional credits through different plans.

# ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/chatnova-ai.git
cd chatnova-ai
2️⃣ Install Dependencies
Client
cd client
npm install
Server
cd server
npm install
3️⃣ Environment Variables

Create a .env file in the server folder.

Example:

PORT=5000
MONGO_URI=your_mongodb_connection
GEMINI_API_KEY=your_gemini_api_key
JWT_SECRET=your_secret
4️⃣ Run the Project

Run backend:

cd server
npm run dev

Run frontend:

cd client
npm start
