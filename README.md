# рџ¤– ChatNova AI

ChatNova AI is a full-stack AI powered chat application that allows users to interact with artificial intelligence for **text generation and image generation**. The platform is built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)** and integrates external AI services like **Google Gemini API** and **ImageKit API**.

The application provides a modern UI with features like **dark mode, credit-based usage system, AI chat, AI image generation, and community images gallery**.
